# StockTransformer模型评估报告

## 1. 执行限制解决方案

### 1.1 PyTorch DLL问题原因
在Windows环境下直接运行Python脚本时，PyTorch的CUDA DLL可能无法正确初始化，导致以下错误：
- `DLL load failed` 或类似DLL初始化错误
- 环境变量配置问题

### 1.2 解决方案
已在`baseline_repo/.venv`中创建完整的虚拟环境，包含：
- Python 3.12
- PyTorch (含CUDA支持)
- CatBoost, LightGBM等机器学习库

**解决方法：**

#### 方法一：使用虚拟环境Python（推荐）
```bash
cd baseline_repo
.venv\Scripts\python.exe your_script.py
```

#### 方法二：设置环境变量后运行
```bash
set TORCH_CUDA_ARCH_LIST=9.0
python your_script.py
```

#### 方法三：强制使用CPU模式
在代码中将device设置为cpu：
```python
device = torch.device('cpu')  # 而不是自动检测
```

## 4. 收益率计算方法

根据`score_self.py`中的定义：

```
收益率 = (end['开盘'] - start['开盘']) / start['开盘']
```

即：最后一条记录的开盘价减去第一条记录的开盘价，除以第一条记录的开盘价。
